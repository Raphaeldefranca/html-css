como usar caracters especiais

negrito

para fazer uma frase contendo um termo em negrito, deve-se usar a tag strong, por questões de semântica

itálico

para utilizar fazer uma frase contendo um termo em itálico, deve-se usar a tag "em" por questões de semântica

texto marcado

podemos criar também um texto marcado usando a tag MARK

textp grande e texto pequeno

estamos criando um texto grande e um texto pequeno nesse parágrafo

texto deletado

podemos marcar um texto como exclúido para indicar que ele deve ser lido, mas não considerado

texto inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois ou meramente sublinhado.

existe também o sublinhado com a tag U (não semântica)

para inserir coisas do tipo x20+3
